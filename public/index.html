Composing App

My most ambitious project yet - I've been developing an Android application for composing music! The app is written in both Java and Kotlin and follows Android best practices: MVVM architecture, unit testing, Material Design, dependency injection, and various design patterns are already implemented in the project. Currently, users can:

  • Add measures to the score and scroll through their piece
  • Change the clef of the score
  • Add notes and rests of the most common note lengths
  • Undo/Redo certain actions

Before fighting with Android's custom Views some more, my next big step for the project is to build a backend so users can save their music.

Big Data Challenge

Each year, Stem Fellowship challenges high-school students and undergraduates to team up and undertake a research project. I wrote a few Python scripts that parsed through JSON and CSV files, then transformed the data with Matlab and Excel into graphs for our report and presentation at nationals.
